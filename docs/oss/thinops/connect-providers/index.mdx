## How to connect

1. **Navigate** to **Data Connections** → **Connect** in NarevAI
2. **Select** your provider from the available cards
3. **Follow** the setup guide for your chosen provider
4. **Test** the connection and start importing data

## Supported Providers

Choose the provider you want to connect:

- **[Amazon Web Services (AWS)](/oss/thinops/connect-providers/aws)** - Cost and Usage Reports & FOCUS Data Exports from S3
- **[Microsoft Azure](/oss/thinops/connect-providers/azure)** - Cost Management Exports in FOCUS format from Blob Storage
- **[Google Cloud Platform (GCP)](/oss/thinops/connect-providers/gcp)** - BigQuery billing exports with FOCUS views
- **[OpenAI](/oss/thinops/connect-providers/openai)** - API usage and billing data via administrator keys

## Data and privacy

✅ **Read-only billing data** - costs, usage, resource metadata  
❌ **No app data** - your workloads and content stay private
